# **App Name**: Canteen Commander

## Core Features:

- Live Stock Summary: Displays a table of inventory with item name, current amount, and threshold. Highlights items in red if stock is below threshold.
- Stock Threshold Alerts: Generates alert cards for low-stock items, providing quick notifications.
- Daily Sales Snapshot: Displays the daily sales count from Firestore, giving an overview of sales performance.
- Most Sold Items Display: Presents a list of the most sold items retrieved from Firestore.
- User Credit Management: Allows inline editing of user credit balances with a direct update to Firestore.
- Order Management: Displays a searchable and filterable list of orders, with order details and timestamps sourced from Firestore.
- Inventory Updates: Allows inline editing of current stock levels and provides a form for adding or removing items, updating Firestore in real-time.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) to provide a comforting and familiar aesthetic.
- Background color: Off-white (#FAFAFA), ensuring a clean and unobtrusive backdrop.
- Accent color: Muted orange (#E07A5F) to draw attention to interactive elements.
- Font: 'PT Sans', a humanist sans-serif which is suitable for both headlines and body text.
- Minimalist, table-based design for clear data presentation.
- Simple, universally recognizable icons for navigation.
- Subtle transitions on data updates to indicate real-time changes.
